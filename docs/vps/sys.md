# 常用命令

## ssh 连接

```
ssh -p [端口号] [用户名]@[IP地址]

```

## 用户与权限管理

### 切换与确认身份

- 切换到 root

```
sudo -i
```

```
sudo su -

```

- 以 root 身份执行单条命令

```
sudo [command]
```

- 查看当前用户

```
whoami
```

- 查看用户 UID/组信息

```
id [用户名]
```

- 查看当前用户所属组

```
groups
```

### 用户管理

- 修改当前用户密码

```
passwd [用户名]
```

- 新建用户

```
adduser [用户名]
```

- 删除用户(包括家目录)

```
userdel -r 用户名
```

- 把用户加入某个组

```
usermod -aG 组名 用户名
```

- 锁定/解锁用户

```
usermod -L 用户名 # 锁定用户
usermod -U 用户名 # 解锁用户
```

- 切换到其他用户

```
su - 用户名
```

### 登录情况查看

- 查看当前登录用户

```
w
```

- 最近登录记录

```
last
```
